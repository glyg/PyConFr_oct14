Le problème biophysique¶

Les dformations du tissus sont dues l'activit d'un ensemble de protines intra et extra cellulaires.

Ces protines forment un maillage entre les cellules, que l'on peut modliser comme un rseau


  • Minimiser localement l'nergie chaque modification
  • quilibre des forces

Le modle est donc bas sur la bibliothque graph_tool crite et maintenue par Tiago P. Peixoto, permettant de manipuler des graphes.

## Let's define a neat plotting function
def show_apopto_surroundings(eptm, a_cell, axes=None):
    eptm.set_local_mask(None)
    eptm.set_local_mask(a_cell, wider=True)
    is_apoptotic = eptm.ixs.copy()
    is_apoptotic.a[:] = 0.
    is_apoptotic[a_cell] = 1.
    ax_zs, ax_xy = lj.plot_2pannels(eptm, axes=axes,
                                    edge_kwargs={'c':'k', 'lw':0.5, 'alpha':0.5},
                                    cell_kwargs={'cell_colors':is_apoptotic,
                                                 'cmap':'Reds',
                                                 'alpha':0.8})

    eptm.set_local_mask(None)
    return ax_zs, ax_xy
